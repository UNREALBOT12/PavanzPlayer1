- Updated Plug-in
- Super Fast Bot
- No Lag Hang
- Fast Download Song From Server
- Program Updated
- Smooth Player

## Commands 🛠

- `/play <song name>` - play song you requested
- `/song <song name>` - download songs you want quickly
- `/reload` - Bot Online OR Offine

#### Admins Only 👷‍♂️
- `/pause` - pause song play
- `/resume` - resume song play
- `/skip` - play next song
- `/end` - stop music play
